# Phishing Email Detection & Awareness System

## Tools Used

* Browser
* Microsoft Word
* GitHub

## Objective

To analyze phishing email samples and identify common phishing indicators used in social engineering attacks.

## Phishing Samples Analyzed

* Fake Fax Message
* Fake Voice Message

## Key Indicators Identified

* Suspicious sender emails
* Fake attachments
* Urgent language
* Social engineering tactics
* Generic greetings

## Recommendations

* Verify sender identity
* Avoid clicking suspicious links
* Do not open unknown attachments
* Enable multi-factor authentication

## Ethical Note

This project was conducted for educational and awareness purposes only.
